# Source Material Dropzone

Place planning and build assets here before implementation:

- profile photos and branding files
- project screenshots (Listegy, Expensior, logistics website)
- legal reference notes and compliance inputs
- any copy references for English and German versions

Suggested structure:

- `assets/source-material/profile/`
- `assets/source-material/listegy/`
- `assets/source-material/expensior/`
- `assets/source-material/logistics-website/`
- `assets/source-material/branding/`
- `assets/source-material/legal/`

Do not commit credentials, tokens, or private client data.
